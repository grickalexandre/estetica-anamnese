Teste do Ícone de Notificação

Este é um teste visual do ícone de notificação que foi implementado no header do sistema.

Como Funciona

  • Ícone Amarelo: Aparece quando há anamneses pendentes
  • Badge Vermelho: Mostra o número de anamneses pendentes
  • Animação: O badge pulsa para chamar atenção
  • Clique: Leva diretamente para a lista de anamneses

Implementação

O sistema foi implementado com:

  • Ícone de sino no PWAHeader.vue
  • Contagem automática via App.vue
  • Comunicação via provide/inject
  • Estilos responsivos e animações
